# Scaffolding
`pollinate right714/base-static --name <PROJECT_NAME>`

# Use
## Start Webserver & SCSS/JS watch
```bash
npm install
npm run dev
```
## Linter
```bash
npm run code-check
```

## Format
```bash
npm run code-format
```

## Build
```bash
npm run build
```
